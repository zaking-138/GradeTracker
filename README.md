Grade Tracker is A desktop application for tracking assignments and grades. The application will include student, teacher, and admin users. Students can view their assignments and record their grades. Teachers using the grade tracker can enter assignment grades, create assignments, and view class statistics. Admin users can create users, courses, and assign teachers to certain tasks.
